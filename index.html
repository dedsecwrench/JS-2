LET vs CONST vs VAR


    Let
  • Let is a Block Scope
  • Let cannot be reinitialized and redeclared
  • Let can be modified later
    Const
  • Const is a Block Scope
  • Const cannot be reinitialized and redeclared
  • Const cannot be modified later
    Var
  • Var is a functional as well as Global Scope
  • Var can be reinitialized and redeclared
  • var can be modified later

It's not a good practise always declare a function first and then execute it. It is a concept which says that you should always declare and initialize all the variables in js before using them. If we don't initialize the variable before using it, it will throw an error. so to avoid such errors we should follow the concept of hoisting.

Temporal Dead Zone

Temporal Dead Zone is the time between the variables hoisted and the variables which gets initialized
